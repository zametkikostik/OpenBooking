# Getting Started with OpenBooking

## Quick Start Guide

This guide will help you get OpenBooking up and running in minutes.

## Prerequisites

Before you begin, ensure you have:

- **Node.js** >= 20.0.0
- **npm** >= 10.0.0
- **Git** for version control

Optional but recommended:
- **Supabase CLI** for local development
- **MetaMask** or similar crypto wallet

## Installation

### Option 1: Automated Setup (Recommended)

```bash
# Clone the repository
git clone https://github.com/zametkikostik/OpenBooking.git
cd OpenBooking

# Run the setup script
chmod +x scripts/setup.sh
./scripts/setup.sh
```

This script will:
- Install all dependencies
- Set up Supabase locally
- Configure environment variables
- Run database migrations

### Option 2: Manual Setup

```bash
# Clone the repository
git clone https://github.com/zametkikostik/OpenBooking.git
cd OpenBooking

# Install dependencies
npm install

# Copy environment file
cp .env.local.example .env.local

# Edit .env.local with your credentials

# Start Supabase locally
supabase start

# Run migrations
supabase migration up

# Start development server
npm run dev
```

## Configuration

### Environment Variables

Edit `.env.local` with your credentials:

```env
# Supabase Local (auto-generated by setup script)
NEXT_PUBLIC_SUPABASE_URL=http://localhost:54321
NEXT_PUBLIC_SUPABASE_ANON_KEY=local-anon-key
SUPABASE_SERVICE_ROLE_KEY=local-service-role

# Ethereum RPC
ETH_RPC_URL=https://eth.llamarpc.com

# Token Addresses (pre-configured)
A7A5_TOKEN_ADDRESS=0x6fA0BE17e4beA2fCfA22ef89BF8ac9aab0AB0fc9
DAI_TOKEN_ADDRESS=0x6B175474E89094C44Da98b954EedeAC495271d0F
```

### Supabase Configuration

For local development:
```bash
supabase start
supabase status  # View local URLs
```

For production:
```bash
supabase login
supabase link --project-ref sibgxcagyylbqmjaykys
supabase db push
```

## Running the Application

### Development Mode

```bash

```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Production Mode


## Troubleshooting

### Common Issues

**Issue**: Supabase not starting
```bash
# Solution: Check Docker is running
docker ps

# Restart Supabase
supabase stop
supabase start
```

**Issue**: Port 3000 already in use
```bash
# Solution: Use different port
PORT=3001 npm run dev
```

**Issue**: Database migrations fail
```bash
# Solution: Reset local database
supabase db reset
supabase migration up
```

**Issue**: Module not found
```bash
# Solution: Clear cache and reinstall
rm -rf node_modules package-lock.json
npm install
```
